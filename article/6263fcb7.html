CC认证标准(GB/T 18336)

发表于2023-04-04|更新于2023-04-05
|阅读量:

0x00 专业术语

  1. 安全保障要求 security assurance requirement(SAR)
    涉及评估对象（TOE）的开发和交付的条件和过程的安全要求，以及评估者对这些条件和过程产生的证据所要求的行为。
  2. 安全功能要求 security function requirement(SFR)
    在特定安全目标（ST）或保护轮廓（PP）中定义的、符合评估对象安全问题定义的安全要求
  3. 安全目标 security target(ST)
    基于安全问题定义，针对评估对象与实现相关的安全要求的陈述
  4. 安全目的 security objective
    用于对抗特定的威胁或满足特定组织安全策略的一种陈述
  5. 安全属性 security attribute
    主体、用户、客体、信息、会话、资源的特征，用于定义安全功能要求
  6. 安全问题定义 security problem definition(SPD)
    以一种形式化的方法定义评估对象要处理的安全基本特征和范围的陈述
  7. 安全要求 security requirement
    在特定安全目标或保护轮廓中定义的一部分评估对象安全规范要求
  8. 保护轮廓 protection profile(PP)
    针对一类评估对象的与实现无关的安全需求陈述

安全目标(ST)的目的：从确定TOE的安全问题定义（SPD）开始，详细的描述要保护的资产以及对这些资产的威胁，然后考虑所有相关的假设后描述TOE需要采取的安全控制措施，并证实这些威胁已被应对。
评估结果的有用性很大程度上取决于安全问题定义（SPD）的质量。
